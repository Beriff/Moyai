No bullshit guide to Moyai

Moyai is a game engine developed by Beriff to be used solely in windows cmd. The engine natively supports raytraced rendering and GUI rendering. Moyai is also a suite of apps to help you develop your games and an interpreter for "moscript", a scripting language designed to be used within moyai applications

Roadmap:
  • GUI rendering
  • Input Handling
  • Raytraced rendering
