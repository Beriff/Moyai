No bullshit guide to Moyai

Moyai is a game engine developed by Beriff to be used solely in windows cmd. The engine natively supports raytraced rendering and GUI rendering. Moyai is also a suite of apps to help you develop your games and an interpreter for "moscript", a scripting language designed to be used within moyai applications

Moyai consists of Moyai Framework, a C# framework, that everything else is built on, Moyai Engine, a game engine built with moyai framework, capable of running .moproj files and Moyai Editor, a .moproj editor which itself is built with the use of moyai framework.

To develop games, you can use Moyai Engine, with the help of the editor, or use the framework directly (which does not implement some features, most notably, the ability to execute .moproj files, but gives more room for optimisation and implementation of additional features)
